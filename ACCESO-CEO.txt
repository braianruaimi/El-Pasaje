🎯 FUNCIONES PRINCIPALES:

🔄 ACTUALIZAR
   • Recarga las métricas sin refrescar página
   • Útil cuando el sitio está abierto en otra pestaña

💾 EXPORTAR DATOS
   • Descarga archivo JSON con toda la información
   • Ideal para backups o análisis externos
   • Formato: elpasaje-metrics-YYYY-MM-DD.json

🏠 IR AL SITIO
   • Vuelve a la página principal del negocio

🗑️ RESETEAR MÉTRICAS
   ⚠️ CUIDADO: Borra TODOS los datos permanentemente
   • Requiere contraseña (1234)
   • Requiere confirmación adicional
   • No se puede deshacer
   • Usar solo al comenzar nueva temporada

🆘 SOLUCIÓN DE PROBLEMAS:

❌ No veo datos
   → Asegúrate de que hubo actividad en el sitio
   → Navega por index.html, agrega productos, haz pedido de prueba

❌ Gráficos no cargan
   → Verifica conexión a internet (usa CDN de Chart.js)
   → Refresca la página (F5)

❌ Datos se borraron
   → Restaura desde último backup JSON exportado
   → Presiona F12 → Application → Local Storage → Restaurar

❌ Métricas no actualizan
   → Click en botón "🔄 Actualizar"
   → Si persiste, refresca página completa
